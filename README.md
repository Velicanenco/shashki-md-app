# Демо-бот "Авто зі США" (Україна)

Рабочий Telegram-бот на aiogram 3, который можно за 5 минут запустить и показать на встрече с потенциальным клиентом (компании, которые перегоняют авто из США в Украину).

## Что внутри

- **🧮 Калькулятор розмитнення** — считает мито (10%), акциз (по объёму двигателя/типу топлива/возрасту) и ПДВ (20%) по формулам, актуальным на 2026 год. Для электро — акциз по ёмкости батареи.
- **🚗 Підбір авто** — короткая анкета (бюджет + тип кузова) → показывает 2-3 подходящих авто из тестовой базы (`MOCK_CARS` в `bot.py`).
- **📝 Заявка** — собирает имя и телефон, сохраняет в `leads.csv` и (если указать `ADMIN_CHAT_ID`) сразу присылает вам в личку.
- **📞 Контакти** и **ℹ️ Як це працює** — статичные разделы, чтобы клиент видел, к кому обращаться и как устроен процесс.

Это ровно та же логика, что в вашем GoPlay/GoSport-хабе: навигация + понятная механика + контакты нужного человека, только под нишу перегона авто.

## Как запустить (5 минут)

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
2. Получите токен бота: напишите в Telegram **@BotFather** → `/newbot` → следуйте подсказкам → скопируйте токен.
3. Скопируйте `.env.example` в `.env` и вставьте токен:
   ```
   cp .env.example .env
   ```
4. Запустите:
   ```
   python bot.py
   ```
5. Откройте вашего бота в Telegram и нажмите **Start**.

Бот можно запустить с любого компьютера или дешёвого VPS (не нужно ничего сложного — обычный long polling, без вебхуков и домена).

## Как быстро адаптировать под конкретного клиента

Всё, что нужно менять под нового клиента, вынесено в начало `bot.py`:

- `COMPANY_NAME` — название компании в приветствии.
- `CONTACTS` — реальные контакты менеджеров клиента.
- `MOCK_CARS` — можно оставить как демо-данные или подключить реальную выгрузку лотов (это уже следующий этап работ, не входит в демо).

Это и есть логика продажи за $300: один раз пишете шаблон (уже готов), а дальше клиенту продаёте по сути настройку + брендинг + подключение реальных данных — быстро воспроизводимо для следующего клиента.

## Важные ограничения демо-версии

- Формулы растаможки — с открытых источников (мито 10%, акциз по объёму/топливу/возрасту, ПДВ 20%). Перед реальным использованием стоит сверить с брокером/актуальным законодательством — таможенные ставки в Украине меняются, и это должно быть явно оговорено с клиентом как "ориентировочный расчёт".
- `MOCK_CARS` — тестовые данные для демонстрации механики подбора, не реальные лоты с аукционов.
- Заявки сохраняются в простой CSV-файл рядом с ботом — для реального использования лучше подключить нормальную БД или CRM, но для демо и первых недель работы этого достаточно.
